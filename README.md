A single-page app (SPA) to learn and play chess online.

### Install and Setup

Create an `.env` file.

```
cp .env.example .env
```

Install the `npm` packages.

```
npm install
```

Add the following entry to your `/etc/hosts` file.

```
127.0.0.1       ui.chesslablab.org
```

Start the app.

```
npm start
```

### Build the App

Build the app for production.

```
npm run build
```
